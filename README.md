willyfog-web
============

Web for the Willyfog project.

## Deploy

```
git clone https://github.com/soutoner/willyfog-web.git
cd willyfog-web
```

1. Do the Vagrant!

```
$ vagrant up
$ vagrant ssh
[...]
```

2. Install dependencies (inside Vagrant): 

```
$ cd ~/willyfog-web
$ composer install
```
